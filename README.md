# 📊 Cereal Dataset Data Analytics

This project analyzes a cereal dataset collected from Kaggle using Python data analytics techniques. The project includes data preprocessing, visualization, and hypothesis testing to discover meaningful insights from cereal nutrition data.

---

## ✨ Features

- Data cleaning & preprocessing
- Handling missing values and duplicates
- Data visualization using charts & heatmaps
- Correlation analysis
- Hypothesis testing using t-test & Pearson correlation
- Statistical insights from cereal nutrition data

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

---

## 📈 Analysis Performed

- Correlation Heatmap
- Histograms
- Boxplots
- Pairplots
- Statistical Analysis
- Hypothesis Testing

---

## 📌 Conclusion

The analysis showed that sugar levels strongly affect calorie content, and hot cereals generally contain more calories than cold cereals.
